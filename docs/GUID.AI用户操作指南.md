## 1. 使用前须知

### 1.1 访问方式

- 主站入口：`index.html`（单页应用，通过页面切换而非独立 URL）。
- Apps 内容在「爆款 Apps」页内以 iframe 加载 `apps.html`。
- 独立分享落地页：`share.html`。

### 1.2 登录状态

| 状态 | 默认进入页面 | 顶栏样式 |
|------|--------------|----------|
| 未登录 | 官网首页 | `#nav-home`（含「登录」） |
| 已登录 | 灵感页（内部路由，本文不展开该页操作） | `#nav-app`（含余额、个人中心） |

登录信息保存在浏览器 `localStorage`（键名 `guid_ai_auth`）。

### 1.3 必须登录才能打开的页面

- **我的产品**（`myassets` / `myassets-mobile`）  
未登录点击时会弹出登录框并提示「请先登录」。

### 1.4 桌面端与移动端差异（总表）

| 功能 | 桌面（≥768px） | 手机（≤768px） |
|------|----------------|----------------|
| 主导航 | 顶栏横向 10 项 | 汉堡菜单 + 底栏 5 项 |
| 个人中心 | 顶栏 👤 | 侧滑菜单底部 |
| 钱包 | 顶栏 ⚡ 余额 | 侧滑菜单底部余额按钮 |
| 我的产品 | `myassets` 瀑布流 | 底栏进入 `myassets-mobile` |
| 底栏 | 不显示 | 图片 / Apps / 我的产品 / 灵感 / 视频 |

底栏五项为快捷入口；**模型、文字、音频、3DGS、Skills** 等需通过顶栏或侧滑菜单进入。

## 2. 全局导航与登录

### 2.1 顶栏导航项（登录后 `#nav-app`）

点击 Logo → 返回**官网首页**。

| 菜单 | 进入页面 | 本文是否详述 |
|------|----------|--------------|
| ✨ 灵感 | `inspiration` | 否 |
| 🤖 模型 | `model-hub` | 是 |
| 💬 文字 | `chat` | 否 |
| 🖼️ 图片 | `image` | 否 |
| 🎬 视频 | `video` | 否 |
| 🎵 音频 | `audio` | 否 |
| 3DGS | `3dgs` | 是 |
| 🎮 Apps | `apps` | 是 |
| 💎 我的产品 | `myassets` | 是 |
| 🔌 Skills | `skills` | 是 |

右侧附加：

- **⚡ 余额胶囊** → 钱包页  
- **👤**（较宽屏幕）→ 个人中心  
- **🌐 语言** → 切换界面语言（演示）

未登录时顶栏为 `#nav-home`，无余额与个人中心，保留「登录」按钮。

### 2.2 手机侧滑菜单

点击顶栏 **☰** 打开 `#header-mobile-menu`：

- 上部：与顶栏相同的页面列表（文案可能更长，如「图片生成」「视频工坊」）。
- **未登录底部**：语言选择 +「登录 / 注册」。
- **已登录底部**：钱包余额按钮 +「个人中心」。

点击遮罩或关闭按钮可收起菜单。

### 2.3 手机底栏（仅 ≤768px）

| 图标 | 页面 | 说明 |
|------|------|------|
| 图片 | `image` | 创作页，本文不展开 |
| Apps | `apps` | |
| 我的产品 | `myassets-mobile` | 需登录 |
| 灵感 | `inspiration` | 本文不展开 |
| 视频 | `video` | 本文不展开 |

打开全屏预览或部分工作区弹窗时，底栏可能自动隐藏以免遮挡。

### 2.4 登录 / 注册弹窗（`login-modal`）

**打开方式**：顶栏「登录」、首页「立即开始」、需登录页拦截、侧滑菜单等。

**操作步骤**：

1. 选择 **手机号** 或 **邮箱** Tab。  
2. 填写账号、验证码；点击 **获取验证码**（演示会倒计时）。  
3. 输入图形验证码（如有）。  
4. 勾选 **用户协议 / 隐私政策**。  
5. 点击 **登录 / 注册** → 约 1.2 秒后关闭弹窗并写入登录态。  
6. 也可点击 **微信 / GitHub / Google / QQ** 第三方登录（演示）。  
7. 点击 **🔒 密码登录** → 进入 `password-login-modal`：账号、密码、图形码、「忘记密码」等。

**密码登录弹窗**：

- 填写账号与密码后登录；忘记密码会 Toast 提示（演示）。

**退出登录**：个人中心底部「退出登录」→ 确认后回到官网首页。

## 3. 官网首页

**页面 ID**：`homepage`  
**典型用户**：未登录访客、点击 Logo 返回的用户。

### 3.1 顶部公告条

| 操作 | 说明 |
|------|------|
| 查看详情 → | 打开 `announcement-modal` 公告摘要 |
| ✕ | 关闭公告条（`closeAnnouncementBar`） |
| 滚动文字 | 自动轮播平台动态（只读） |

### 3.2 Hero 主视觉区

| 操作 | 说明 |
|------|------|
| 👉 立即开始免费创作 | 打开登录弹窗 |
| 🔍 探索灵感画廊 | 进入灵感页（访客也可浏览，本文不展开） |
| 打字机文案 | 自动切换能力卖点（只读） |

### 3.3 核心功能矩阵

- 展示五大创作方向卡片（灵感 / 文字 / 图片 / 视频 / 音频）。  
- 未登录点击多数能力会引导登录或跳转对应页（跳转目标本文不展开）。

### 3.4 模型展示横滑区

- 横向滚动浏览热门模型卡片。  
- 点击可进入模型广场或触发登录（以实际按钮为准）。

### 3.5 灵感案例区

- 横向案例卡片 → 通常进入灵感相关页面（本文不展开）。

### 3.6 页脚链接

常见入口：模型广场、API 文档、Skills、平台公告、帮助等 → 对应 `showPage` 页面。

## 6. 爆款 Apps

**页面 ID**：`apps`（内嵌 `apps.html`）

### 6.1 页面结构

| 区域 | 操作 |
|------|------|
| 顶部搜索 | 输入关键词过滤应用名称/描述 |
| 精选应用 | 横向大卡，点击「立即使用」 |
| 图片精选 | 图像类官方/精选应用网格 |
| 影音精选 | 视频/音频类应用 |
| 文案精选 | 文案类应用 |
| 精选工具 Bento | 工具型应用拼图布局 |
| 探索区 | Tab（全部/图像/视频/音频等）+ 瀑布流 |

### 6.2 打开应用

1. 点击任意应用卡片。  
2. 父页面接收 `postMessage`，调用 `openAppTemplateModal`。  
3. 根据应用类型打开：  
   - **图像类工作区**（全屏 `omni-image-workspace-modal` 等）  
   - **文案工作区**（`app-copy-workspace-modal`）  
   - 或简单模板弹窗 `app-template-modal`

### 6.3 图像类工作区弹窗（通用结构）

适用于人物替换、旧照时光机、360 全景图、电商详情页等。

**左侧栏**

| 元素 | 操作 |
|------|------|
| 标题 / 播放数 | 只读 |
| 标签 | 只读 |
| ♥ 收藏 / ★ 评论数 | 切换收藏、查看互动数（演示） |
| API / 工作流 | 跳转或 Toast（演示） |
| 创作者 + 关注 | 点击「+ 关注」切换关注态 |
| 参数折叠面板 | 点击标题栏展开/收起；内可上传图片、填 prompt 等 |
| 上传槽 | 点击上传、✎ 编辑、× 删除 |
| 底部 **运行** | 消耗算力并生成（演示）；部分应用显示 **⚡0.7** 绿色运行条 |

**主内容区**

| 元素 | 操作 |
|------|------|
| 应用介绍 & 输入建议 | 点击右侧箭头折叠/展开 |
| Banner / 海报 | 左右切换官方样例（1/n） |
| 作品 / 评论 Tab | 切换列表 |
| 作品卡片 | 点击可打开灵感详情叠加层（部分应用） |

**关闭**：点击弹窗右上角 ✕ 或遮罩（以实现为准）。

### 6.4 移动端

- 通过底栏 **Apps** 直达。  
- iframe 高度随窗口调整；工作区弹窗可能全屏并隐藏底栏。

## 8. 我的产品（桌面版）

**页面 ID**：`myassets`（**需登录**）

### 8.1 顶部 Tab

| Tab | 内容 |
|-----|------|
| 我的作品 | 个人生成历史（图/视/音/文） |
| 我的发布 | 已发布到广场的内容 |
| 我的收藏 | 收藏的应用或作品 |

点击 Tab 切换；右侧 **管理** 进入批量管理模式。

### 8.2 我的作品 — 卡片操作

鼠标悬停或查看卡片底部工具条：

| 图标 | 功能 |
|------|------|
| 分享 | 打开 `myworks-share-modal` |
| 发布 | `showEditProduct` 打开发布编辑 |
| 再次生成 | `regenerateProduct` 用相同参数重跑 |
| 下载 | Toast 提示下载（演示） |
| 删除 | `deleteProduct` 删除该条记录 |

**点击卡片主体** → 打开 `myworks-preview-modal` 大屏预览（支持左右切换、信息面板）。

**加载更多** → 列表底部按钮追加历史（演示）。

### 8.3 管理模式下

- 再次点击 **管理** 退出。  
- 卡片可出现勾选框，支持批量删除等（以界面为准）。

### 8.4 我的发布 Tab

| 操作 | 说明 |
|------|------|
| 搜索框 | `filterMyWorks()` 按关键词过滤发布 |
| 发布卡片 | 查看、编辑、下架等（演示） |

### 8.5 我的收藏 Tab

- 展示收藏的应用/作品卡片。  
- 点击 **Apps 类收藏**（如旧照时光机、360 全景图）→ 直接打开对应全屏工作区。  
- 点击设计/漫剧/音乐类 → 进入对应详情页（见第 21 节）。

### 8.6 底部快捷入口

通常包括：**钱包**、**交易记录**、**我的作品**子页、**我的收藏**子页等。

### 8.7 相关弹窗

| 弹窗 | 用途 |
|------|------|
| `myworks-publish-modal` | 填写标题、描述后发布 |
| `myworks-share-modal` | 复制链接、生成分享海报 |
| `myworks-preview-modal` | 全屏预览与快捷操作 |

## 14. API 密钥管理

**页面 ID**：`api`

| 操作 | 说明 |
|------|------|
| + 新建密钥 | `create-key-modal` → 填写名称/权限 → `create-key-success-modal` 展示密钥（仅一次） |
| 调用统计 | 今日 / 周 / 月 / 累计（只读） |
