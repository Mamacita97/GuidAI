什么是多模型协作？
多模型协作是GUID.AI的核心功能之一，它可以让多个顶尖AI模型同时回答同一个问题，然后由总结模型将各模型的回答整合成一份更全面、更可靠的答案。
为什么使用多模型协作？
• 多元视角：不同模型有不同的训练数据和侧重点
• 交叉验证：减少单一模型的幻觉和错误
• 最优答案：AI自动分析对比，生成最佳答案
• 并行加速：多模型同时运算，极速响应
如何使用？
1. 进入聊天页面
2. 点击「⚡多模型协作」按钮
3. 选择参与回答的模型（最多5个）
4. 选择总结模型
5. 输入问题，点击发送
6. 等待所有模型回答完成，查看整合结果
适用场景
• 复杂问题分析
• 多角度方案对比
• 重要决策支持
• 代码审查与优化

❓ 常见问题（FAQ）
如何注册GUID.AI账号？ ▼
GUID.AI采用「登录即注册」的方式，无需单独注册：

1. 点击官网首页的「登录」按钮
2. 选择「手机登录」或「邮箱登录」
3. 输入手机号/邮箱，点击「获取验证码」
4. 输入验证码，点击「登录/注册」
5. 系统自动创建账号，直接进入灵感广场

💡 提示：首次登录后，建议在个人中心完善个人信息和绑定第三方账号。
忘记密码怎么办？ ▼
如果您忘记密码，可以通过以下方式重置：

1. 在登录页面点击「忘记密码」
2. 输入注册时使用的手机号或邮箱
3. 获取验证码并验证身份
4. 设置新密码

💡 新密码建议使用8位以上字母+数字组合。
如何切换模型？ ▼
切换模型非常简单：

1. 在聊天/图片/视频/音频页面找到模型选择器
2. 点击下拉菜单
3. 从列表中选择你需要的模型
4. 系统会自动切换，无需重新输入

💡 不同模型的功能和价格可能不同，请根据需求选择。
积分可以退款吗？ ▼
关于积分退款政策：

• 未使用的积分可申请退款
• 已使用的积分不支持退款
• 活动赠送积分不支持退款
• 退款将通过原支付渠道返回

💡 如有退款需求，请通过「联系客服」提交申请。
提现需要多久到账？ ▼
提现到账时间：

• 支付宝：1-3个工作日
• 微信支付：1-3个工作日
• 银行卡：3-7个工作日

提现审核通过后，平台会尽快处理您的提现申请。
Skills功能如何使用？ ▼
Skills功能让AI Agent具备图像/视频生成能力：

1. 进入Skills页面
2. 复制MCP服务器配置
3. 将配置添加到你的MCP客户端（Claude/Cursor/Codex等）
4. 在对话中通过MCP协议调用GUID.AI的功能

💡 详细配置方法请查看Skills页面的文档。

📖 简介

MCP（Model Context Protocol）是一个开放标准，允许AI代理访问外部工具。GUID.AI通过MCP协议，让您的AI Agent具备图像和视频生成能力。

✅ 支持30+顶尖AI模型
✅ 图片/视频/音频全能力
✅ 无需API密钥，账号认证即可
✅ 与平台积分体系打通
⚙️ 安装配置

将以下MCP配置添加到你的Agent客户端配置文件中：

{
  "mcpServers": {
    "guid": {
      "url": "https://api.guid.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
🚀 第一个请求

配置完成后，在Agent中直接说："用GUID生成一张赛博朋克图片"

Agent会通过MCP协议调用GUID.AI的图像生成API，返回生成的图片。

💬 对话API
端点：https://api.guid.ai/v1/chat/completions
方法：POST
curl -X POST https://api.guid.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "GPT-5.4",
    "messages": [{"role": "user", "content": "你好"}]
  }'
响应示例：
{
  "choices": [{
    "message": {"role": "assistant", "content": "你好！有什么我可以帮助你的吗？"}

}
