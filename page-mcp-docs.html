📖 简介

MCP（Model Context Protocol）是一个开放标准，允许AI代理访问外部工具。GUID.AI通过MCP协议，让您的AI Agent具备图像和视频生成能力。

✅ 支持30+顶尖AI模型
✅ 图片/视频/音频全能力
✅ 无需API密钥，账号认证即可
✅ 与平台积分体系打通
⚙️ 安装配置

将以下MCP配置添加到你的Agent客户端配置文件中：

{
  "mcpServers": {
    "guid": {
      "url": "https://api.guid.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
🚀 第一个请求

配置完成后，在Agent中直接说："用GUID生成一张赛博朋克图片"

Agent会通过MCP协议调用GUID.AI的图像生成API，返回生成的图片。

💬 对话API
端点：https://api.guid.ai/v1/chat/completions
方法：POST
curl -X POST https://api.guid.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "GPT-5.4",
    "messages": [{"role": "user", "content": "你好"}]
  }'
响应示例：
{
  "choices": [{
    "message": {"role": "assistant", "content": "你好！有什么我可以帮助你的吗？"}
  }]
}

🐍 Python SDK
pip install guid-ai-sdk

from guid_ai import GUIDClient
client = GUIDClient(api_key="YOUR_API_KEY")
response = client.chat.completions.create(
    model="GPT-5.4",
    messages=[{"role": "user", "content": "你好"}]
)
📦 JavaScript SDK
npm install guid-ai-sdk

import { GUIDClient } from 'guid-ai-sdk';
const client = new GUIDClient({ apiKey: "YOUR_API_KEY" });
const response = await client.chat.completions.create({
    model: "GPT-5.4",
    messages: [{ role: "user", content: "你好" }]
});
❓ 常见问题
支持哪些Agent？▼
支持 Claude Web、Claude Code、Cursor IDE、Codex、OpenClaw Agent、Hermes Agent、NemoClaw 等任何支持MCP协议的客户端。
费用如何计算？▼
使用平台积分（⚡算力）计费：图片生成⚡0.06-0.24/次，视频生成⚡0.35-2.80/次，对话按token计费。
需要API密钥吗？▼
需要在API令牌页面创建密钥，配置到MCP配置文件中进行身份验证。
