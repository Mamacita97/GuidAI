✅ 任何支持 MCP 协议的客户端均可连接

🔧 MCP 服务器配置 
{
  "mcpServers": {
    "guid": {
      "url": "https://api.guid.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

💡 将以上配置添加到 MCP 客户端配置文件中

❓ 常见问题
如何连接到 GUID.AI？ ▼
GUID.AI 使用 MCP 开放标准。在「我的」→ 密钥管理中创建 API Key → 写入 MCP 配置 → 在对话中调用即可。 前往密钥管理
哪些 Agent 支持？ ▼
Claude Web / Code、Cursor、Codex、OpenClaw、Hermes、NemoClaw，及所有支持 MCP 的客户端。
费用如何计算？ ▼
与平台共用 ⚡ 算力：图片 ⚡0.06–0.24/次，视频 ⚡0.35–2.80/次，对话按 token 计费。
生成需要多长时间？ ▼
图片约 5–30 秒，视频约 1–5 分钟，音频约 10–60 秒，视负载而定。
需要 API 密钥吗？ ▼
需要。在API 令牌页面创建后填入 MCP 配置。
